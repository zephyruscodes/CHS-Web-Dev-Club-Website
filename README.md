# CHS Web Development Club Website

Welcome to the official website repository for the CHS Web Development Club! This is a modern, responsive web platform showcasing our club's mission, projects, and community.

## 🌟 Features

- **Modern Design**: Dark-themed interface with animated gradients and smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Cursor-following light effects and hover animations
- **Smooth Navigation**: Seamless scroll navigation between sections
- **Tailwind CSS**: Utility-first CSS framework for rapid styling
- **Vanilla JavaScript**: No heavy dependencies – pure JavaScript for interactivity

## 📋 Sections

The website includes the following sections:

- **Home**: Eye-catching hero section with club motto "The Syntax of Success"
- **About**: Information about why students should join the club
- **Projects**: Gallery of club member projects
- **Leaders**: Meet the club leadership team
- **Contact**: Get in touch with us

## ✨ Key Features Explained

### Animated Gradient Title
The main title features a smooth color-shifting animation that cycles through blue, pink, and purple.

### Cursor-Following Light
A custom light effect that follows your mouse cursor across the page, creating a dynamic visual experience.

### Smooth Scrolling
Navigation links smoothly scroll to their target sections with proper offset for the fixed navigation bar.

### Responsive Design
The website adapts beautifully to different screen sizes:
- Mobile-first approach
- Flexible grid layouts
- Responsive typography

## 🎨 Customization

### Colors
Colors are defined in the CSS. Look for color classes like `text-club-blue` and `text-club-purple` to customize the color scheme.

### Content
Edit the text, headings, and descriptions directly in `index.html` to update club information.

### Images
Replace placeholder images in the `images/` directory with your own club photos.
